🧠 Cheat Sheet UI – Color, Tipografía y Layout

Guía práctica para tomar decisiones de diseño visual claras, accesibles y coherentes.

🎨 Color

✔️ ¿Qué hacer?
  • Usa 1 color primario dominante + 1 o 2 secundarios.
  • Colores neutros (grises, blanco, negro) para fondo y texto.
  • Asegura contraste suficiente: texto claro sobre fondo oscuro (mín. 4.5:1).
❌ Errores comunes
  • Usar muchos colores sin jerarquía clara.
  • Colores que no contrastan: gris claro sobre blanco = ilegible.
  • Comunicar solo con color (ej: usar solo rojo para mostrar error).
🎯 Consejo: Los colores también transmiten emociones. Azul = confianza, Verde = éxito, Rojo = alerta.

🔤 Tipografía

✔️ ¿Qué hacer?
  • Usa solo 1 o 2 familias tipográficas (ej.: Montserrat + Roboto).
  • Jerarquiza con tamaño, peso y color (no subrayes).
  • Texto legible: cuerpo mínimo 16px, interlineado 1.5×.
❌ Errores comunes
  • Usar fuentes decorativas o con serifas finas en móvil.
  • Justificar texto: rompe el ritmo de lectura.
  • Colocar texto centrado en párrafos largos.
📐 Consejo: Usa alineación izquierda en bloques de texto. Usa tamaños progresivos (32px, 24px, 18px, 16px…).

📐 Layout (Distribución)

✔️ ¿Qué hacer?
  • Usa márgenes amplios y espacio en blanco para respirar (mínimo 16px).
  • Organiza el contenido por secciones lógicas y alineadas.
  • Rejilla flexible (ej.: 12 columnas) para mantener consistencia visual.
❌ Errores comunes
  • Todo del mismo tamaño = no hay jerarquía.
  • Textos o botones desalineados entre sí.
  • Elementos demasiado pegados sin aire visual.
🧩 Consejo: Usa patrones conocidos: cabecera–contenido–pie, tarjetas, menú lateral, etc.

📱 Responsive y Adaptabilidad

✔️ ¿Qué hacer?
  • Diseña primero para móvil (mobile first).
  • Botones grandes (mínimo 48x48px).
  • Elimina contenido innecesario en pantallas pequeñas.
💡 Ejemplo real: En móvil solo muestro el botón “Reservar libro” y oculto filtros avanzados.

🔄 Comparativa: diseño tradicional vs. UI moderna

  • Tradicional: diseño rígido, poco accesible, poco enfocado en la experiencia.
  • UI moderna: jerarquía visual, adaptabilidad, simplicidad, enfoque en el usuario.
